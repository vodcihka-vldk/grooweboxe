# Архитектурная документация GrooveBoxe

Этот каталог содержит полную архитектурную документацию для проекта GrooveBoxe - аппаратного groovebox sampler на базе ESP32.

## Структура документации

```
docs/
├── ARCHITECTURE.md          # Общая архитектура системы
├── hardware/                # Спецификации компонентов
│   └── SPECIFICATIONS.md    # Технические характеристики
├── software/                # Программная архитектура
│   └── ESP32_ARCHITECTURE.md # Архитектура ESP32 прошивки
├── protocols/               # Коммуникационные протоколы
│   └── COMMUNICATION.md     # Форматы сообщений и протоколы
└── api/                     # API документация
    └── API_REFERENCE.md     # Интерфейсы и функции
```

## Документы

### 1. ARCHITECTURE.md
Общая архитектура системы, включая:
- Архитектурные слои
- Компоненты системы (аудио, интерфейс, хранение, коммуникации)
- Программную архитектуру
- Архитектурные паттерны
- Требования к производительности
- Безопасность и надежность

### 2. hardware/SPECIFICATIONS.md
Технические спецификации компонентов:
- Микроконтроллер ESP32-S3
- Аудио система (DAC, усилитель)
- Дисплеи (OLED, TFT)
- Ввод данных (энкодеры, кнопки)
- Хранение данных (SD карта, Flash)
- Питание и электрические характеристики
- Механические характеристики
- Температурный режим
- ЭМС и сертификация

### 3. software/ESP32_ARCHITECTURE.md
Программная архитектура ESP32 прошивки:
- Структура исходного кода
- Главный модуль и инициализация
- Аудио подсистема (движок, каналы, эффекты)
- Секвенсор
- Пользовательский интерфейс
- Система хранения
- Драйверы периферии
- Архитектурные паттерны
- Оптимизации производительности
- Отладка и тестирование
- Безопасность и надежность

### 4. protocols/COMMUNICATION.md
Коммуникационные протоколы:
- WebSocket Bridge (веб-прототип ↔ ESP32)
- UART Protocol (ESP32 ↔ внешние устройства)
- I2C Protocol (управление периферией)
- SPI Protocol (SD карта, дисплей)
- I2S Protocol (аудио передача)
- Обработка ошибок и таймауты
- Отладочный интерфейс

### 5. api/API_REFERENCE.md
API интерфейсы:
- Audio API (аудио движок, каналы, эффекты)
- Sequencer API (секвенсор, паттерны)
- UI API (дисплей, ввод, меню)
- Storage API (файловая система, проекты)
- Driver API (драйверы периферии)
- Utility API (утилиты, математика, отладка)
- Обработка ошибок
- Примеры использования
