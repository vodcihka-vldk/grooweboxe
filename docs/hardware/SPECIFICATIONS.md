# Спецификации компонентов GrooveBoxe

## Основные компоненты

### 1. Микроконтроллер

**ESP32-S3-WROOM-1-N8**
- Процессор: Xtensa LX7, 240MHz
- SRAM: 512KB + 384KB
- PSRAM: 8MB (внешняя)
- Flash: 8MB
- Интерфейсы: I2S, SPI, I2C, UART, USB OTG
- Рабочее напряжение: 3.3V

**Альтернатива: ESP32-S3-WROOM-2-N8**
- Встроенный кристалл 40MHz
- Встроенный LDO

### 2. Аудио система

**I2S DAC: PCM5102A**
- Разрядность: 32-bit
- Частота дискретизации: до 216kHz
- SNR: 112dB
- THD+N: -90dB
- Напряжение питания: 3.3V - 5V

**Альтернатива: ES8388**
- Stereo codec
- Встроенный PGA
- Поддержка DSP эффектов
- I2C управление

**Аудио усилитель: LM4871**
- Выходная мощность: 1W (8Ω)
- Напряжение питания: 2.2V - 5.5V
- Класс AB

### 3. Дисплей

**OLED: SSD1306 128x64**
- Интерфейс: I2C/SPI
- Размер: 0.96"
- Напряжение: 3.3V
- Контрастность: высокая
- Угол обзора: 160°

**TFT: ILI9341 320x240**
- Интерфейс: SPI 4-wire
- Размер: 2.8"
- Цветность: 16-bit (65K цветов)
- Напряжение: 3.3V

### 4. Ввод данных

**Энкодеры: EC11**
- Пульсации: 24 PPR
- Нажатие: встроенный переключатель
- Напряжение: 5V/3.3V
- Долговечность: 20000 циклов

**Кнопки: Tactile Switch**
- Размер: 6x6mm
- Сила нажатия: 160gf
- Долговечность: 500000 циклов
- Напряжение: 12V DC

### 5. Хранение данных

**MicroSD карта**
- Объем: до 2GB (FAT16)
- Скорость: Class 10
- Интерфейс: SPI
- Напряжение: 3.3V

**SPI Flash (встроенный)**
- Объем: 8MB
- Интерфейс: QIO
- Скорость: 80MHz

### 6. Питание

**USB-C разъем**
- Напряжение: 5V
- Ток: до 500mA
- Защита: TVS диод

**LDO: AMS1117-3.3**
- Выходное напряжение: 3.3V
- Ток: 1A
- Защита: от перегрева и перегрузки

**Конденсаторы**
- Развязка: 100nF керамические
- Фильтрация: 100μF электролитические
- Аудио: 220μF tantalum

## Электрические характеристики

### Потребление тока

**ESP32-S3:**
- Активный режим: 80-120mA
- Сон: 5-10μA
- Deep Sleep: 2.5μA

**Аудио система:**
- DAC: 10-20mA
- Усилитель: 50-200mA (в зависимости от громкости)

**Дисплей:**
- OLED: 10-20mA
- TFT: 30-50mA

**Общее потребление:**
- Без аудио: 50-80mA
- С аудио: 150-300mA

### Требования к питанию

**Напряжение:**
- ESP32: 3.3V ±10%
- Аудио: 3.3V ±5%
- Дисплей: 3.3V ±10%
- Кнопки: 3.3V

**Стабильность:**
- Пульсации: < 50mVpp
- Переходные процессы: < 100ms
- Защита от обратной полярности
